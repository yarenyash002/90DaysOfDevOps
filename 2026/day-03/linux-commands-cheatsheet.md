## Process management

1.ps : it will show the process   
2.ps -a : it will show the active process   
3.top : check memory usage, check all running process  
4.ps -ef : shows snapshot of all running processes in full format   
5.nohup & : its character in the end of line, which in the background
6.kill pid : kill the process for particular id


## File system

1.chown:change the permission of the file  
2.chmod:change the permission of directory or modify the directory  
3.wc: check the word count  
4.cp:copy the file  
5.cat: shows the content of the file  
6.pwd: present work directory  
7.head -f:shows the first 10 lines of file  
8.tail -f: shows the last 10 lines of file  
9.du -sh: show the size of file/folder  
10.df -h: display free disk  

## Networking troubleshooting
1.ping : to check the network of remote server    
2.curl :command line to transfer data to or from server usig url  
3.ipaddr: to find the ip address  
4.ifconfig : display  the ip address name  
5.hostname -f : shows the name of hostname  
6.uname -r : show the version of os  
7.traceroute : see the route packets take to reach a host  
8.netstat : check the open ports and active connections
